# 📊 Análise de Vendas com Pandas

## 📖 Sobre o projeto

Este projeto representa meu **primeiro projeto prático na área de Ciência de Dados**, desenvolvido para aplicar os conhecimentos iniciais adquiridos em Python e durante meus estudos por meio dos cursos da **Data Science Academy**.

A aplicação realiza a leitura de um arquivo CSV contendo informações de vendas e, utilizando a biblioteca **Pandas**, processa os dados para gerar análises importantes. Além disso, emprega o **Matplotlib** para criar uma visualização gráfica dos resultados.

Este projeto marca o início da minha jornada na área de dados e demonstra, na prática, conceitos fundamentais de manipulação e análise de informações.

---

## 🚀 Tecnologias utilizadas

- Python 3
- Pandas
- Matplotlib

---

## 📈 Funcionalidades

- 📄 Leitura de dados a partir de um arquivo CSV.
- 💰 Cálculo automático do faturamento de cada venda.
- 📊 Identificação do produto mais vendido.
- 🏷️ Identificação da categoria com maior faturamento.
- 💵 Cálculo do faturamento total.
- 📅 Descoberta do dia com maior faturamento.
- 📉 Cálculo do ticket médio das vendas.
- 📊 Geração de gráfico de barras com o faturamento por categoria.
- 💾 Exportação dos resultados para um novo arquivo CSV.

---

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/LetiMonteiro/Analise-Vendas.git
```

Entre na pasta do projeto:

```bash
cd Analise-Vendas
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o programa:

```bash
python analise.py
```
